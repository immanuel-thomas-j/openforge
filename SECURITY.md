# Security Policy

We take security vulnerabilities seriously at **OpenForge**. We are committed to protecting our contributors, users, and the integrity of the application. 

This document outlines our policy for reporting security vulnerabilities. Please review it before submitting any security-related reports.

---

## Supported Versions

Currently, we only actively support and patch the latest version of the repository on the `main` branch. 

| Version | Supported |
| ------- | --------- |
| Main (ECSoC 2026) | ✅ Yes |
| All prior versions | ❌ No |

---

## Reporting a Vulnerability

**DO NOT open a public GitHub Issue for security vulnerabilities.** Publicly disclosing a security flaw before a patch is ready exposes the application to potential exploits.

If you believe you have discovered a security vulnerability in OpenForge, please report it privately:

1. **Email the Project Admin:** Send a detailed email to **immanuelthomasj10@gmail.com** (or contact the Project Admin directly via their GitHub profile contact info).
2. **Include Details:** Please include as much information as possible to help us reproduce and resolve the issue:
   - A clear description of the vulnerability and its potential impact.
   - Step-by-step instructions or a proof-of-concept (PoC) script to reproduce the issue.
   - Any suggestions for a fix or mitigation.
3. **Keep it Confidential:** We ask that you keep all information regarding the vulnerability confidential until we have had a reasonable amount of time to analyze, patch, and deploy a fix.

---

## Our Response Process

After receiving a security report, the Project Admin will:

1. **Acknowledge:** Acknowledge receipt of the vulnerability report within **48 hours**.
2. **Evaluate:** Evaluate the severity and impact of the reported vulnerability.
3. **Patch:** Work on a patch to resolve the issue as quickly as possible.
4. **Notify:** Notify you once the patch has been developed and merged.
5. **Credit:** With your permission, we will gladly credit you in our release notes or security patch announcements for your responsible disclosure!

Thank you for helping keep OpenForge secure!
